# Core Files

经过和颖奇讨论，这个主要是应用程序Crash的时候，会自动保存堆载信息，便于调试；

## 参考文档

* [Core Files on the Raspberry Pi](https://docs.faircom.com/doc/knowledgebase/76189.htm)
* [详解coredump](https://blog.csdn.net/MOU_IT/article/details/88903668)
